Assessment on Mutillidae Web Application

Completed a penetration testing project as part of the Cybersecurity Academy Training by the National Telecommunication Institute (NTI). The project focused on assessing security vulnerabilities in Mutillidae, a deliberately vulnerable web application.

Key Deliverables:

  • Identified and exploited multiple vulnerabilities including Local File Inclusion (LFI), Cross-Site Scripting (XSS), SQL Injection, Cross-Site Request Forgery (CSRF), Command Injection & Reverse Shell, PII Disclosure, IDOR, and Security Misconfigurations.
  • Documented findings in a professional penetration testing report with Proof of Concepts (PoCs).
  • Applied industry-standard tools such as Burp Suite, OWASP ZAP, and Kali Linux for testing.
  • Demonstrated ability to **Bypass authentication** and **Expose or modify database contents** using SQL injection payloads.
